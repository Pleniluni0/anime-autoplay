Reproducción automática
Al terminar un episodio aparece un contador con cuenta atrás. Pulsa Ver ahora para saltar inmediatamente o Cancelar para quedarte. Si no haces nada, pasa solo al siguiente episodio.
Recordar reproductor
Guarda qué servidor estás usando (Tamamo, Voe, Daily…). Al pasar al siguiente episodio selecciona ese servidor automáticamente.
Modo automático
Al pasar de episodio, el host nativo simula un click real en el centro de la pantalla. Esto activa el reproductor y la pantalla completa sin que tengas que tocar nada.
Requiere activación del host: el script Python debe estar instalado y registrado en Windows. Ejecuta install_host.ps1 con el ID de la extensión una sola vez para activarlo.

Si el host no está disponible, el overlay seguirá apareciendo pero necesitarás pulsar tú manualmente.
Activar el host nativo
1. Instala la dependencia: pip install pyautogui
2. Copia el ID de la extensión desde chrome://extensions/
3. Ejecuta en PowerShell desde la carpeta host\:
.\install_host.ps1 -ExtensionId "tu_id_aqui"
4. Reinicia Chrome. Listo.
Saltar créditos / ending
El overlay de cuenta atrás aparece N segundos antes del final del vídeo. Útil para saltar el ending y pasar directo al siguiente episodio. Ponlo a 0 para que aparezca cuando termine el vídeo.
Saltar intro / opening
Configura los tiempos de inicio y fin de la intro (ej: 0:00 → 1:30).
• Manual: aparece un botón "⏩ Saltar intro" sobre el vídeo.
• Automático: salta solo sin mostrar ningún botón.
Consejo: Los tiempos son globales para todos los episodios. Si la intro cambia de duración entre temporadas, ajústalo aquí.
Cuenta atrás
Segundos que espera antes de pasar al siguiente episodio. Entre 3 y 15 segundos.
